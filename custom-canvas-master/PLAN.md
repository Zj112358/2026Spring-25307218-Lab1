# Custom Canvas 持续演进计划

> 最后更新：2026-05-26 | 版本：v1.0.0

---

## 一、项目现状概览

| 维度 | 详情 |
|------|------|
| 项目类型 | HarmonyOS 5 原生画布应用（ArkTS + ArkUI） |
| 核心架构 | 双层 Canvas 增量渲染（O(1) 性能） |
| 已实现功能 | 3种笔刷、16种图形、双模式选色、撤销/重做、选择/移动/旋转/删除、双指缩放、折叠屏适配、国际化 |
| 未完成功能 | 铅笔/激光笔（字符串已定义但未实现）、圆规圆弧（状态变量已声明但未集成到触摸流程）、选择工具入口 |
| 测试覆盖 | 无 |
| 第三方依赖 | 无 |

## 三、Phase 1 — 基础补全（优先级：🔴 高）

> 目标：补全已有资源定义但未实现的功能，消除半成品状态

### 1.1 圆规圆弧功能集成
- **现状**：`Index.ets` 中 `compassCenterFixed`/`compassCenterX`/`compassCenterY`/`compassStartAngle`/`compassDrawing` 已声明，但未接入触摸交互流程
- **任务**：
  - [ ] 实现圆规第一次 TouchDown+Move 画完整圆的逻辑
  - [ ] 实现第二次 TouchDown+Move 画圆弧的逻辑（支持起止角度）
  - [ ] 在作图工具面板添加圆规按钮 UI
  - [ ] 集成 ShapeDraw 的 startAngle/endAngle 属性
- **涉及文件**：`Index.ets`、`IDraw.ets`（ShapeDraw）
- **预估工作量**：2-3天

### 1.2 铅笔笔刷实现
- **现状**：国际化字符串 `pencil` 已定义（zh_CN/en_US），代码中未实现
- **任务**：
  - [ ] 在 `IBrush.ets` 中实现 `PencilBrush` 类（模拟铅笔质感：纹理边缘+压力模拟）
  - [ ] 在 `myPaintSheet.ets` 画笔面板添加铅笔选项
  - [ ] 在 `Index.ets` 添加铅笔状态管理逻辑
- **涉及文件**：`IBrush.ets`、`myPaintSheet.ets`、`Index.ets`
- **预估工作量**：1-2天

### 1.3 激光笔笔刷实现
- **现状**：国际化字符串 `laser_pointer` 已定义，代码中未实现
- **任务**：
  - [ ] 在 `IBrush.ets` 中实现 `LaserPointerBrush` 类（发光高亮线条+淡出动画）
  - [ ] 在画笔面板添加激光笔选项
  - [ ] 实现激光笔淡出效果（定时清除最近N秒的笔画）
- **涉及文件**：`IBrush.ets`、`myPaintSheet.ets`、`Index.ets`
- **预估工作量**：2天

### 1.4 选择工具 UI 入口
- **现状**：`select_tool` 字符串已定义，但无独立 UI 按钮，当前通过点击已有路径触发
- **任务**：
  - [ ] 在工具栏添加"选择"按钮（与画笔/作图/橡皮擦并列）
  - [ ] 选择模式下点击空白区域取消选择
  - [ ] 优化选中对象的视觉反馈（边框+控制点）
- **涉及文件**：`Index.ets`
- **预估工作量**：1天

## 八、架构演进方向

### 当前架构
```
Index.ets (上帝组件：1169行)
  ├── myPaintSheet.ets (画笔面板)
  └── viewmodel/
       ├── DrawInvoker.ets (命令管理)
       ├── IBrush.ets (笔刷)
       ├── IDraw.ets (绘制对象)
       └── Paint.ets (绘制属性)
```

### 目标架构
```
pages/Index.ets (页面壳，路由+布局)
  ├── components/
  │    ├── CanvasArea.ets        (双层Canvas渲染区)
  │    ├── ToolBar.ets           (底部工具栏)
  │    ├── BrushPanel.ets        (画笔设置面板)
  │    ├── ShapePanel.ets        (作图工具面板)
  │    ├── LayerPanel.ets        (图层面板)
  │    └── SelectHandler.ets     (选择/变换控制器)
  ├── viewmodel/
  │    ├── DrawInvoker.ets       (命令管理)
  │    ├── LayerManager.ets      (图层管理)
  │    ├── SelectionManager.ets  (选区管理)
  │    ├── ExportManager.ets     (导出管理)
  │    ├── brush/
  │    │    ├── IBrush.ets
  │    │    ├── NormalBrush.ets
  │    │    ├── FountainPenBrush.ets
  │    │    ├── PencilBrush.ets
  │    │    └── LaserPointerBrush.ets
  │    ├── draw/
  │    │    ├── IDraw.ets
  │    │    ├── DrawPath.ets
  │    │    ├── ShapeDraw.ets
  │    │    └── TextDraw.ets
  │    └── Paint.ets
  └── common/
       ├── CommonConstants.ets
       └── ColorUtils.ets
```

## 十一、如何使用本计划

1. **选择当前阶段**：从 Phase 1 开始，按顺序推进
2. **勾选任务**：完成一项后将 `[ ]` 改为 `[x]`，并更新"最后更新"日期
3. **新增任务**：在对应 Phase 下新增条目，标注优先级
4. **跨阶段调整**：根据实际需求，可将后续 Phase 的任务提前
5. **定期回顾**：建议每两周回顾一次本计划，调整优先级和里程碑
6. **版本发布**：完成一个 Phase 后，更新版本号和更新日志
